# Feature Inventory

> Comprehensive list of functional requirements for the new Astro-based platform.

## Core Features

- **Responsive Mobile-First Design:** Fully adaptive layout using Tailwind CSS v4, optimized for mobile browsing by busy executives.
- **Sticky Navigation Header:** Persistent top navigation with a primary "Contact" Call-to-Action button.
- **Dynamic Practice Area Hubs:** Dedicated layouts for "Commercial Litigation," "Collections," etc., with sidebar navigation for sub-services.
- **Attorney Profile System:** Structured bio pages for Michael Baim, Patricia Lepak, and Brian Berlin with vCard integration.
- **Secure Contact Form:** Netlify-ready form with fields for case type and description, including spam protection.
- **Content Management System (MDX):** File-based CMS for easily updating blog posts, news, and practice area content.
- **SEO Architecture:** Automated sitemap generation, canonical tags, and Open Graph social sharing cards.
- **Schema Markup:** JSON-LD injection for `LegalService`, `Attorney`, and `LocalBusiness` data.
- **Performance Optimization:** Image optimization (WebP/AVIF) and code splitting via Astro.
- **Client Portal Link:** External link capability in the utility nav for the secure payment gateway.

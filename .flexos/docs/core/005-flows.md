# Flows & Business Logic

> Critical user journeys designed to convert visitors into legal clients.

## 1. Visitor to Lead Conversion Flow
1.  **Entry:** User lands on "Commercial Collections" page via organic search.
2.  **Education:** User reads about "Creditors' Rights" and "Asset Discovery."
3.  **Trust:** User sees "Law Office Automation" badge and attorney credentials.
4.  **Action:** User clicks sticky "Request Consultation" button or scrolls to footer form.
5.  **Submission:** User fills out Name, Company, and Case Description.
6.  **Confirmation:** System redirects to `/thank-you` and triggers email notification to `contactEmail`.

## 2. Blog Discovery Flow
1.  **Entry:** User searches "Breach of contract litigation Illinois."
2.  **Consumption:** User reads blog post "What Chicago Businesses Need to Know About Breach of Contract Law."
3.  **Cross-Sell:** Sidebar displays "Related Practice Area: Commercial Litigation."
4.  **Transition:** User clicks through to the service page.
5.  **Conversion:** User engages with the consultation form on the service page.

## 3. Mobile Navigation Flow
1.  **State:** Mobile user lands on Homepage.
2.  **Action:** Taps "Hamburger" menu icon.
3.  **Transition:** Drawer slides out from right.
4.  **Decision:** User expands "Practice Areas" accordion.
5.  **Selection:** User taps "Real Estate" -> Navigation drawer closes -> Page loads.
